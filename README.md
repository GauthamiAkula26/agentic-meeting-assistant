# 🧠 Agentic Meeting Assistant

An AI-powered meeting intelligence application that transforms raw meeting transcripts into structured decisions, tracked action items, searchable memory, and exportable follow-ups.

## Features

- ✅ Extract decisions, action items, and risks
- ✅ Identify owners and due dates
- ✅ Track task status
- ✅ Search across multiple meetings
- ✅ Ask AI questions over meeting memory
- ✅ Export Jira-ready CSV and markdown summaries
- ✅ Premium SaaS-style UI

## Tech Stack

- Streamlit
- Python
- BM25 retrieval with `rank-bm25`
- OpenAI (optional)
